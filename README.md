# Avi's Compiler

A web-based IDE that allows you to write and execute code in various languages.

## Supported Languages

*   Web Development (HTML, CSS, JavaScript)
*   Python
*   Java
*   C++
*   C#
*   Ruby
*   Go
*   Rust

## Running the Project

1.  **Prerequisites:** Make sure you have Node.js and the necessary compilers for the languages you want to use installed on your system. You can refer to the `Dockerfile` for a complete list of dependencies and installation instructions.
2.  **Install dependencies:** `npm install`
3.  **Start the server:** `node server.js`
4.  Open your browser and navigate to `http://localhost:3000`.
